# WetoCode 0.2.4

## 本次更新

- 品牌定位调整为“更符合中国开发者使用习惯的中文桌面 Coding Agent”。
- 移除界面、CLI、项目说明和 Agent 身份中的行业限定标签。
- 标题栏副标题调整为“中文桌面 Coding Agent”。
- 上下文面板将行业规则调整为通用的项目安全规则。

- 自动迁移旧版行业规则文件，不影响用户的模型配置、会话和其他设置。
