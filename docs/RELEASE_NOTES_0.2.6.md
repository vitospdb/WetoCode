# WetoCode 0.2.6

## 本次更新

- 修复 Windows 终端可能继承宿主名称并显示 `Cursor 219` 等标题的问题。
- 桌面集成 CLI 和独立 `wetocode` 命令现在都会保持进程标题、Windows 控制台标题和标准终端标题为 `WetoCode`。
- 集成终端继续显示 `WetoCode >`，保留普通项目 Shell 和危险操作权限确认。
