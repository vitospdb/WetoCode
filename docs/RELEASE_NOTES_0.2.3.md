# WetoCode 0.2.3

## 本次更新

- 集成终端默认进入安装包内置的 WetoCode CLI，无需另行安装 CLI 或 OpenCode。
- 可在 WetoCode CLI 与项目 Shell 之间直接切换。
- 终端移到当前任务标题下方并提高可用高度。
- 当前任务右上角可直接切换已配置的模型。
- 界面缩放改为 80% 到 140% 的连续滑杆。
- 左上角副标题调整为“智能开发工作台”。
- 安装时选择父目录后自动追加 `WetoCode`，例如 `D:\Program Files` 会安装到 `D:\Program Files\WetoCode`。
- Windows 安装包使用原生 NSIS 构建，修复旧版本卸载器完整性检查失败的问题。

## 下载

- `WetoCode-Setup-0.2.3-x64.exe`：Windows x64 安装版。
- `WetoCode-Portable-0.2.3-x64.zip`：Windows x64 免安装版。
- `SHA256SUMS-0.2.3.txt`：下载文件完整性校验值。

当前 Windows 安装包尚未进行商业代码签名，系统可能显示“未知发布者”。
