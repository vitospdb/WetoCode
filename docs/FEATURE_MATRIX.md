## 完成定义

“已实现”必须同时满足：界面存在、主进程/引擎协议真实接通、权限边界明确、自动化测试或真实 Electron 冒烟验证通过。仅有按钮、静态演示数据或未接通服务的入口不计为实现。

本地自动化在应用运行或系统托盘驻留时调度，不等同于云端离线任务。开发预览只允许受限开发命令和本机回环地址，不等同于已获得用户浏览器身份的 Chrome 扩展或远程 Computer Use。
