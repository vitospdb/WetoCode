# WetoCode 0.2.8

## 本次更新

- WetoCode CLI 首页提示、命令面板、会话操作、权限对话框和常见状态改为中文显示。
- 修复 Windows 中文输入法无法向集成终端提交文字的问题，并补充中文字体回退。
- 集成终端新增右键“复制 / 粘贴”菜单，同时支持 `Ctrl+Shift+C` 和 `Ctrl+Shift+V`。
- 保留真实交互 TUI、WetoCode 品牌、项目 Shell 和原有权限边界。
- 新增真实 Electron 回归测试，覆盖中文输入法、中文剪贴板粘贴、终端中文界面和 Shell 切换。
