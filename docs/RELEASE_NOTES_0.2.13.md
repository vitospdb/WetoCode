# WetoCode 0.2.13

## 本次更新

- 桌面端顶部“选择任务模型”直接列出 OpenCode 公共免费模型，点击即可切换。
- 内置 Big Pickle、DeepSeek V4 Flash、HY 3、MiMo V2.5、Nemotron 3 Ultra 和 North Mini Code 作为离线兜底目录。
- 打开项目后从 OpenCode 实时模型目录同步，并只显示当前可用且官方输入、输出价格均为零的模型。

## 下载

- `WetoCode-Setup-0.2.13-x64.exe`：Windows x64 安装版。

当前 Windows 安装包尚未进行商业代码签名，系统可能显示“未知发布者”。
