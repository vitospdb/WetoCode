# WetoCode 0.2.5

## 本次更新

- 集成终端改用 WetoCode 自有交互层，显示 `WetoCode >`，不再显示底层执行引擎品牌。
- 支持直接输入开发任务，并提供 `/help`、`/new`、`/clear`、`/exit` 命令。
- 危险操作仍在 WetoCode CLI 中进行权限确认，不因界面替换而放宽安全边界。
- 保留普通项目 Shell 模式和内置执行引擎，无需用户额外安装 CLI。
- 内置模型改名为“公共免费模型”，明确其为第三方公共试用服务。
- 安装包不携带发布者个人 API Key；未登录试用额度按公网 IP 统计，同一网络可能共享限额。
- 用户自行配置的模型和内网网关继续使用各自系统密钥环中的凭据。

## 下载

- `WetoCode-Setup-0.2.5-x64.exe`：Windows x64 安装版。
- `WetoCode-Portable-0.2.5-x64.zip`：Windows x64 免安装版。
- `SHA256SUMS-0.2.5.txt`：下载文件完整性校验值。

当前 Windows 安装包尚未进行商业代码签名，系统可能显示“未知发布者”。
